# 行为准则

## 我们的承诺

我们致力于为所有参与者提供无骚扰的协作环境，不因年龄、体型、残障、族裔、性别认同、经验水平、国籍、外貌、种族、宗教或性取向而区别对待。

## 期望行为

- 保持尊重、友善、建设性沟通。
- 以善意提出与接收反馈。
- 以项目与社区共同利益为优先。
- 对他人保持同理心。

## 不可接受行为

- 骚扰、歧视、仇恨言论。
- 人身攻击、恶意挑衅、侮辱性言论。
- 未经许可公开他人隐私信息。
- 其他在专业场景中不当的行为。

## 执行

项目维护者负责解释与执行本准则，并可对不当行为采取适当措施。

## 举报

如遇行为问题，请通过私下渠道联系维护者。

维护者将尽快、公平地处理。
